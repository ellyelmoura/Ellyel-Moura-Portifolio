Minhas Habilidades

HTML

HTML é uma linguaguem de marcação utilizada para construções de páginas web que podem ser interpretadas por navegadores.

CSS

CSS é a maneira de adicionar estilos a página criada em HTML. Dessa forma podemos manipular a formatação, fazer edições de cores, texto e ajustar todos os elementos dentro da página visando a melhor experiência do usuário.

JavaScript

JavaScript é uma linguagem de programação client-side, interpretada, geralmente utilizada com HTML e CSS para desenvolvimento web. Dessa forma, ao utilizar o JavaScript, tornamos as interações com os usuários possíveis.
